Wireless Display Dongle

Raspberry Pi -pohjainen langaton näyttödongle — AirPlay + NDI

Konsepti

Yksi dongle, kaksi protokollaa. Automaattinen vaihto NDI:n (4K) ja AirPlayn (1080p) välillä.

MacBook ──AirPlay 1080p──→ Wi-Fi ──→ Dongle ──HDMI──→ Näyttö Mac Mini ──NDI 4K────────→ Wi-Fi ──→ Dongle ──HDMI──→ 4K Näyttö (valinnainen HDMI→DVI -adapteri DVI-näytöille)
Lite-profiili
1080p

AirPlay Pi Zero 2 W

MacBook → natiivi AirPlay-peilaus. Ei lisäohjelmistoja Maciin.

4K-profiili
3840×2160

NDI Pi 4

Mac Mini → NDI Screen Capture. brew install --cask ndi-tools

Toimintalogiikka

┌───────────────────────────────────────────┐ ▼ │ IDLE │ (etsi NDI:tä) │ │ │ NDI löytyy? ──Kyllä──→ NDI 4K ──────katkeaa────→─┘ │ │ Ei │ │ │ AirPlay kuuntelee ──NDI löytyy──→ NDI 4K │ │ │ asiakas yhdistää │ │ │ AirPlay aktiivinen ──────────katkeaa────────────→─┘ (ei etsi NDI:tä)
  • IDLE: Etsi NDI-lähdettä ensin (4K priorisoitu)
  • Ei NDI:tä: AirPlay kuuntelee, mutta etsii NDI:tä samalla
  • AirPlay aktiivinen: Ei etsi NDI:tä — ei häiritä yhteyttä
  • Yhteys katkeaa: Palaa IDLE → etsi NDI ensin

Asennus

Dongle (Pi)

1
Flashaa Raspberry Pi OS Lite (Bookworm) SD-kortille Raspberry Pi Imagerilla. Aseta Wi-Fi ja SSH.
2
SSH sisään ja suorita:
sudo ./setup/install.sh "Olohuone"
3
sudo reboot — dongle on valmis käyttöön.

Skripti tunnistaa automaattisesti Pi-mallin ja säätää asetukset (GPU-muisti, ylikellotus, swap, jäähdytys).

Mac Mini (4K NDI -lähde)

1
brew install --cask ndi-tools
2
Käynnistä NDI Screen Capture → valitse näyttö → 4K

MacBook (AirPlay)

1
Ohjauskeskus → Näytön peilaus → valitse donglen nimi
